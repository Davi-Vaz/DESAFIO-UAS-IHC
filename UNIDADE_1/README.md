## UA1 – Introdução à IHC

### Objetivo

O objetivo desta atividade foi propor melhorias na interface de uma aplicação de gerenciamento de folha de pagamento, visando aperfeiçoar a **navegabilidade** e resolver as críticas dos clientes relacionadas à dificuldade de uso, baixa conversão e erros de grafia, conforme o cenário apresentado no desafio oficial.

### Solução Desenvolvida

A solução desenvolvida focou na reprogramação da interface de uma janela de confirmação de exclusão de usuários. As principais melhorias implementadas foram:

- **Rótulos de Ação Claros:** Substituição de botões genéricos como "Avançar" por "Confirmar" e "Cancelar", tornando as ações mais intuitivas.  
- **Feedback Visual e Hierarquia:** Utilização de cores (vermelho para "Confirmar") para destacar a criticidade da ação e guiar a atenção do usuário.  
- **Personalização e Contexto:** Substituição da palavra "usuário" pelo nome e ID reais da pessoa, reduzindo a ambiguidade e o risco de erros.  
- **Controles de Navegação:** Adição de um botão "X" para fechar a janela e inversão da ordem dos botões ("Confirmar" à direita, "Cancelar" à esquerda) para seguir padrões de usabilidade.  
- **Legibilidade:** Melhoria da fonte para otimizar a visualização geral da interface.

### Como Executar e Visualizar a Solução

Para visualizar a solução proposta nesta unidade, siga o passo a passo abaixo:

#### 1. Ferramentas Necessárias

Para uma melhor experiência, recomendamos o uso do **Visual Studio Code (VS Code)**, que é um editor de código gratuito. Você pode baixá-lo em: [code.visualstudio.com](https://code.visualstudio.com/).

#### 2. Passo a Passo

1. **Baixe os arquivos:** Certifique-se de ter baixado a pasta `UNIDADE_1` para o seu computador.  
2. **Abra o VS Code:** Inicie o programa Visual Studio Code.  
3. **Abra a pasta:** No VS Code, vá em `Arquivo` > `Abrir Pasta...` e selecione a pasta `UNIDADE_1`.  
4. **Visualize o HTML:**  
   - Localize o arquivo `Desafio_UA1.html` na barra lateral esquerda.  
   - Clique com o botão direito sobre ele e selecione **"Open with Live Server"** (se você tiver essa extensão instalada) ou simplesmente localize o arquivo na pasta do seu computador e dê um clique duplo para abri-lo no seu navegador (Chrome, Edge ou Firefox).
