## UA3 – Storyboarding e prototipação de interfaces

### Objetivo
O objetivo desta atividade foi projetar uma solução computacional para auxiliar Ricardo, um usuário com deficiência visual progressiva, a se locomover em ambientes desconhecidos, apresentando o passo a passo do processo de desenvolvimento, conforme o caso apresentado no desafio oficial.

### Solução Desenvolvida
O trabalho envolveu a equipe de design da empresa Desenvolve! para projetar uma solução que auxiliasse Ricardo. O processo de desenvolvimento seguiu as seguintes etapas:

- **Compreensão do Problema:** Primeiro, a equipe conversou com Ricardo para avaliar o nível de perda de visão, verificar quais dispositivos ele já usa e identificar o que dificulta sua locomoção em lugares desconhecidos. Com isso, foi criada uma persona baseada no perfil do cliente, que ajudou a guiar as decisões do projeto e a manter toda a equipe alinhada sobre as necessidades da persona.
- **Definição da Solução:** Como Ricardo já usa um smartphone e um relógio inteligente, a solução definida foi a criação de um aplicativo com navegação por voz. O aplicativo fornece instruções de áudio em tempo real para guiar Ricardo em ambientes desconhecidos.
- **Prototipação Evolutiva:** Em seguida, foram criados protótipos simples para validar o fluxo de interação. Os protótipos foram evoluídos progressivamente até se aproximarem do produto final.
- **Testes com o Usuário:** Por fim, foram realizados testes com Ricardo para verificar o que funciona e o que precisa ser ajustado antes de iniciar o desenvolvimento completo.
